Osiris - a Revolt selfbot

Osiris is a selfbot for Revolt (revolt.chat), built using Node.js and websockets, made to enhance the user experience!

Features

Message Encryption

Currently has 42 commands!

Installation

- Osiris Download Link

To install Osiris, you need to install the packages first which are listed in modules.txt. After that, you need to add your credentials in the config.json file. When you're done, you just need to run the Osiris.js file! I hope you'll enjoy the selfbot me and my team made :)
